# Sinatra & MVC

* Deliverables:
    
    * View all the books
        * get /books
    * View info on an individual book
        * get /books/:id
    * Create a new book
        * get /books/new
        * post /books
    * edit an existing book
        * get /books/:id/edit
        * patch /books/:id
    * delete an existing book
        * delete /books/:id
